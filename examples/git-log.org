* git log

Show a history of commits.

** Show a history of commits:

#+BEGIN_SRC sh
  git log
#+END_SRC

** Show the history of a particular file or directory, including differences:

#+BEGIN_SRC sh
  git log -p ${path}
#+END_SRC

** Show only the first line of each commits:

#+BEGIN_SRC sh
  git log --oneline
#+END_SRC
